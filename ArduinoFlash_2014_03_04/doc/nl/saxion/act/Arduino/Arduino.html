Arduino()Constructor
public function Arduino(pinConfiguration:Array, tcpPort:uint = 5331, host:String = localhost)

Constructor. You must specify the pinNr configuration and can specify the tcp port

Parameters
pinConfiguration:Array — Array with pinNr configuration, first two are null, others are { pwmOut or digitalIn or digitalOut or servo }
 
tcpPort:uint (default = 5331) — The tcp port to listen to, see also the as2arduinoGlue configuration file
 
host:String (default = localhost) — the host on which the serialproxy port can be found.
