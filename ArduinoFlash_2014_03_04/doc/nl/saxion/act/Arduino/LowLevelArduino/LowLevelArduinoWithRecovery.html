Classpublic class LowLevelArduinoWithRecovery
InheritanceLowLevelArduinoWithRecovery LowLevelArduino flash.events.EventDispatcher

The LowLevelArduino class sends and receives commands according to the Firmata V2.2 protocol. Works with a Arduino and the StandardFirmata firmware. It remembers the state of the pins, input and output values. Based on ideas from Erik Sjodin, eriksjodin.net Bjoern Hartmann, bjoern.org, Mochammad Effendi (Arduino Mega) and Kasper Kamperman

public function LowLevelArduinoWithRecovery(pinConfiguration:Array, tcpPort:uint = 5331, host:String = localhost, isDisableReportingOnConnect:Boolean = true)

Constructor. You must specify the pin configuration and can specify the tcp port and if it is an Arduino Mega

Parameters
pinConfiguration:Array — Array with pin configuration, first two are null, others are { pwmOut or digitalIn or digitalOut or servo }
 
tcpPort:uint (default = 5331) — The tcp port to listen to, see also the as2arduinoGlue configuration file
 
host:String (default = localhost) — Whether or not this Arduino is a Mega arduino with more in/outputs.
 
isDisableReportingOnConnect:Boolean (default = true)
