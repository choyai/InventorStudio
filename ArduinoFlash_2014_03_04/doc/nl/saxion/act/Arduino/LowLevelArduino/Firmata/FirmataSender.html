Classpublic class FirmataSender
InheritanceFirmataSender Object

The FirmataSender class sends commands according to the Firmata V2 protocol. Works with an Arduino and the StandardFirmata firmware. Based on ideas from Erik Sjodin, eriksjodin.net Bjoern Hartmann, bjoern.org and Mochammad Effendi (Arduino Mega)
