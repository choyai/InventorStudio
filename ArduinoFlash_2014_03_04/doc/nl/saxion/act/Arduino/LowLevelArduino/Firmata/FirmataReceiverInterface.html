Interfacepublic interface FirmataReceiverInterface
Implementors LowLevelArduino

The FirmataReceiverInterface shows all callback functions for all firmata messages The FirmataCommandReceiver uses these function to call a specific receiver Based on ideas from Erik Sjodin, eriksjodin.net Bjoern Hartmann, bjoern.org and Mochammad Effendi (Arduino Mega)
